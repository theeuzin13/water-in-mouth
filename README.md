### 1. Migrations

> - npm run migration:generate --name=NomeDaMigration (criar uma migration)
> - npm run migration:run (executar migrations)
> - npm run migration:rever (reverter a última migration)

### 2. Seeds

> - npm run seed (Rodar as seeds para DB. Obs.: Isso só funciona por conta das configs no package.json)

### 3. Seeds Manually

> - npm run seed:admin
> - npm run seed:product
